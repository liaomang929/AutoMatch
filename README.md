# AutoMatch - 足球赛事智能分析工具

一个面向足球竞彩分析师的本地化工具，集成赛事数据抓取、智能选场、AI辅助分析、公众号/直播文案生成等功能。

## 功能特性

### 🎯 五大核心模块
1. **赛事数据抓取** - 从500彩票网抓取官方竞彩足球数据
2. **智能选场与预测** - 自动推荐重点比赛，支持分析师预测录入
3. **AI辅助分析** - 多模型支持（智谱GLM-4/OpenAI兼容接口），生成专业赛事分析
4. **热门文案生成** - 公众号推文（800-1200字）+ 直播文案（三场比赛版，毒舌导演分析师风格）
5. **模型配置管理** - 在线配置AI模型参数，支持多模型切换

### 🔧 技术栈
- **前端**: React + Vite + Ant Design (Antd)
- **后端**: Node.js + Express
- **数据抓取**: puppeteer-core（使用系统Chrome）
- **AI服务**: 智谱GLM-4（默认）/ OpenAI兼容接口（DeepSeek、通义千问、Moonshot等）/ 自定义接口
- **数据存储**: 本地文件系统（JSON + Markdown格式）
- **违禁词过滤**: 自动过滤敏感词，直播文案使用独特的"核心暗语对照"

## 🚀 快速开始

### 环境要求
- Node.js v18+
- Google Chrome浏览器（puppeteer-core依赖）
- 智谱GLM-4 API Key 或其他支持的AI服务API Key

### 安装与运行
```bash
# 1. 克隆仓库
git clone <repository-url>
cd autoMatch

# 2. 安装后端依赖
npm install

# 3. 安装前端依赖
cd client
npm install
cd ..

# 4. 启动后端服务（端口3001）
npm run dev

# 5. 另开终端，启动前端服务（端口5173）
cd client
npm run dev
```

### 访问地址
- **前端界面**: http://localhost:5173
- **后端API**: http://localhost:3001
- **API健康检查**: http://localhost:3001/api/health

## 📋 使用流程

1. **配置AI模型**
   - 访问 http://localhost:5173
   - 进入"模型配置"页面，配置AI模型参数
   - 测试连接确保AI服务可用

2. **获取比赛数据**
   - 进入"赛事数据"页面
   - 点击"一键抓取"获取当日比赛数据

3. **选场与预测**
   - 进入"选场预测"页面
   - 系统自动推荐4-6场重点比赛
   - 录入分析师预测（胜平负、信心指数、分析笔记）

4. **AI分析生成**
   - 进入"AI分析"页面
   - 批量生成或单场生成专业赛事分析文案

5. **文案生成**
   - 进入"文案生成"页面
   - 生成公众号推文（最热1场，800-1200字）
   - 生成直播文案（最热3场，每场200-300字，毒舌导演分析师风格）

## 🔧 配置说明

### AI模型配置
通过"模型配置"页面在线配置，支持三种模式：
1. **智谱GLM-4** - 默认模式，需提供API Key
2. **OpenAI兼容接口** - 支持DeepSeek、通义千问、Moonshot等
3. **自定义接口** - 可配置任意HTTP兼容接口

### 数据存储
所有数据按日期存储在本地目录：
`~/Desktop/AutoMatch/{YYYY-MM-DD}/`
- `01_原始数据/` - 抓取的比赛数据
- `02_重点比赛/` - 选中的重点比赛及预测
- `03_AI分析/` - AI生成的赛事分析
- `04_公众号文案/` - 公众号推文
- `05_直播文案/` - 直播文案

## 🎨 特色功能

### 直播文案生成（毒舌导演分析师风格）
- 每天选取3场比赛，每场200-300字分析
- 使用独特的"核心暗语对照"避开违禁词：
  - 赔率/水位 → 热度/温度
  - 让球/盘口 → 设定/剧情要求
  - 下注/博彩 → 跟剧/押注
  - 收割/杀红 → 摘果子/收场
  - 强队/弱队 → 主角/配角
- 语气笃定、毒舌、揭秘感，适合直播提词器

### 违禁词过滤
- 后端双重过滤：AI Prompt约束 + bannedWords.js词库替换
- 50+敏感词自动替换为合规表达
- 直播文案使用独特的隐喻映射
